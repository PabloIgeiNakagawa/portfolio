# Portfolio - Pablo Igei Nakagawa

Portfolio personal desarrollado con React y TypeScript, donde comparto mis proyectos y experiencia como desarrollador de software.

---

## Características

- **Diseño responsive** - Adaptado para todos los dispositivos
- **Modo claro/oscuro** - Toggle automático según preferencias del sistema
- **Animaciones fluidas** - Transiciones suaves con GSAP
- **Proyectos interactivos** - Galería con videos y descripciones detalladas
- **SEO optimizado** - Meta tags, Open Graph y Twitter Cards
- **Performance** - Lazy loading de imágenes y optimización de assets

## Inicio rápido

```bash
# Clonar el repositorio
git clone https://github.com/PabloIgeiNakagawa/portfolio.git

# Instalar dependencias
npm install

# Iniciar desarrollo
npm run dev

# Verificar build
npm run build
```

## Estructura del proyecto

```
src/
├── assets/           # Imágenes, videos e iconos
├── components/       # Componentes reutilizables (Navbar, Footer, etc.)
├── layout/           # Layout principal de la aplicación
├── pages/            # Páginas del portfolio (Home, Proyectos, Contacto)
│   └── home/
│       ├── components/
│       │   └── hero/
│       │       ├── Hero.tsx
│       │       ├── HeroData.ts
│       │       └── ButtonSocial.tsx
│       ├── About.tsx
│       ├── Technologies.tsx
│       ├── Projects.tsx
│       └── Contact.tsx
├── App.tsx           # Componente principal y rutas
└── main.tsx          # Punto de entrada
```
